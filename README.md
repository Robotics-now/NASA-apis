# NASA-apis
## <a href="https://apod-app.streamlit.app](https://nasa-apod-api.streamlit.app"> Astronomy picture of the day website </a>

This website uses Python with the Streamlit library to create the UI. You can visit the website by following the link above. The APOD, title, and explanation are received via NASA's APOD API. You can also view old APODs by visiting the Previous APODs page. Links to all resources are below:


- `Streamlit documentation` → <a href="https://docs.streamlit.io"> Streamlit </a>
- `Nasa API documentation` → <a href="https://api.nasa.gov"> NASA Open APIs </a>
- `Python code` → <a href="https://github.com/Robotics-now/NASA-apis/blob/main/Today.py"> Github </a>
